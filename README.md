## My process

### Built with

- Semantic HTML5 markup
- CSS custom properties
- CSS Grid and Flexbox
- Mobile-first responsive workflow
- Vanilla JavaScript for the share toggle
- Accessibility improvements for color contrast and button states

### What I learned

- How to build a responsive card component that changes layout between mobile and desktop.
- How to create an interactive share menu using JavaScript, `aria-expanded`, and the `hidden` attribute.
- How small color changes can improve accessibility and help text pass WCAG contrast checks.

### Continued development

- Add the final deployed links after publishing the project.
- Keep practicing accessible interactive components like menus, popovers, and toggles.
- Spend more time comparing finished builds against the design to improve visual accuracy.
